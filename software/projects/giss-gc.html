GISS-GC¶

Coupling effort for the GISS Model E weather and climate model with the GEOS-Chem atmospheric chemistry model. Both models are written in Fortran.

Project GitHub repo: https://github.com/fetch4/GISS-GC.

Main contribution¶

I led ICCS’ effort to complete the coupling effort initiated in GCAP 2.0. This involved project management, as well as the technical tasks of configuring, building and running the coupled model on different high-performance computers, setting up development tools for the project, and ensuring that restart files and diagnostics are handled correctly.

As part of this project, I had a contribution included in GEOS-Chem version 14.5.1 and have an upcoming contribution to version 14.6.0.

Project information¶

This work was done as part of my work as an RSE at the Institute of Computing for Climate Science (ICCS). It was a Virtual Earth Science Research Institute (VESRI) FETCH4 project funded by Schmidt Sciences.
