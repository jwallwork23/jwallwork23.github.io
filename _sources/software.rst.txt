Software
========

Projects I currently work on:

* `Animate <https://github.com/mesh-adaptation/animate/>`__ (lead developer):
  Python-based anisotropic mesh adaptation toolkit for Firedrake.
* `Movement <https://github.com/mesh-adaptation/movement/>`__ (lead developer):
  Python-based mesh movement toolkit for Firedrake.
* `Goalie <https://github.com/mesh-adaptation/goalie/>`__ (lead developer):
  Python-based goal-oriented error estimation and mesh adaptation tool for
  Firedrake.
* `FTorch <https://cambridge-iccs.github.io/FTorch/>`__ (core developer):
  A library for coupling PyTorch models to Fortran.
* `GISS-GC <https://github.com/fetch4/GISS-GC>`__ (core developer):
  Coupling an atmospheric chemical transport model and a general circulation
  model, both written in Fortran.
* `UKCA <https://www.ukca.ac.uk/wiki/index.php/UKCA>`__ (developer):
  Fortran-based atmospheric chemistry model.
* `PSyclone <https://github.com/stfc/psyclone>`__ (contributor):
  Python-based domain-specific compiler and code transformation system for
  Earth-system models written in Fortran.

Projects I worked on in the past:

* `nextSIM-DG <https://github.com/nextsimhub/nextsimdg>`__ (developer):
  Next-generation discontinuous Galerkin sea ice model written in C++.
* `Thetis <https://thetisproject.org/>`__ (core developer):
  Python-based unstructured mesh discontinuous Galerkin coastal ocean model
  built upon Firedrake.
* `Firedrake <https://firedrakeproject.org/>`__ (developer):
  Python-based finite element package which automatically generates C kernels
  and uses PETSc to solve linear and nonlinear systems.
* `PETSc <https://www.mcs.anl.gov/petsc/>`__ (contributor):
  C-based solver library.
