nextSIM-DG I/O
==============

`nextSIM-DG <https://github.com/nextsimhub/nextsimdg>`__ is a next-generation
discontinuous Galerkin sea ice model written in C++.

Main contribution
^^^^^^^^^^^^^^^^^

Enabling writing of files using the XIOS I/O tool.

Funding information
